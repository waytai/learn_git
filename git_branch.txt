创建一个自己的远程分支
1.首先切换到一个本地分支
git checkout liu

* liu
  master
  tian
  wayne

  remote/origin/HEAD -> orgin/master
  remote/origin/master
  remote/origin/wayne



2.推送本地分支到远程分支
git push origin liu

* liu
  master
  tian
  wayne

  remote/origin/HEAD -> orgin/master
  remote/origin/liu
  remote/origin/master
  remote/origin/wayne

3.删除一个远程分支
git push origin :liu

4.重命名本地仓库
git branch -m liu wayne

5.暂时保存，便于切换本地分支
git stash

git stash list

git stash pop stash@{0}



6.回退到某个版本
git reset --hard 77360d054632159a4e7408ed23e77ee06cf0eb2e


7.回退到基本版本
git reset --hard HEAD^

8.与前面版本比较
git diff HEAD^ 
git diff HEAD~3
